Saucedemo UI Automation Tests

Проект с автоматизированными UI-тестами авторизации для сайта:

https://www.saucedemo.com/

Тестовое задание по автоматизации тестирования на Java.

Технологический стек:

- Java 17
- Maven
- Selenium WebDriver
- JUnit 
- Allure Reports
- WebDriverManager
- Page Object Model (POM)

______

Инструкция по запуску:

-Клонируем проект

git clone https://github.com/Leksa-Git/saucedemo-autotests.git

-Переходим в папку

cd saucedemo-autotests

-Тест через Maven

mvn clean test

-Генерация Allure отчета:

allure serve target/allure-results
